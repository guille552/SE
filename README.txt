ARQUITECTURA DE UN  SISTEMA EXPERTO

OBJETIVO: Diseñar la arquitectura de un sistema experto en Prolog o en Python

INTRODUCCIÓN: La estructura de un Sistema experto está organizada alrededor de 3 elementos principales: Base de conocimiento, Base de hechos y Motor de inferencia. Además, se requiere de otros módulos como el de Adquisición, el de Explicación y las Interfaces, que aseguran el diálogo entre el hombre y la máquina en las distintas etapas de la vida de un SE desde su diseño y desarrollo hasta su ejecución y mantenimiento (aprendizaje), sin olvidar que en su creación es común utilizar bases de datos existentes y/o sensores según la aplicación final del SE.
